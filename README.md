# Prashant Portfolio

Single-page portfolio website for Prashant Kr Prasad (Backend Developer & ML Enthusiast).

## Run locally

Just open `index.html` in your browser.

## Deploy

- **GitHub Pages**: push these files to a repo, then enable Pages on the `main` branch.
- **Vercel / Netlify**: import the repo, select this folder as a static site (no build step needed).
